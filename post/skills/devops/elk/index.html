常用于 数据的采集 过滤，配置文件使用 Ruby 编写

  1. input 数据源 通过 input plugin 可支持 JDBC Kafka 文件流 HTTP 等等
  2. filter 数据源做处理 类似于 Java8 中 Stream 表达式 中的map filter等函数，对数据进行处理
  3. ouput 输出数据源 对应 output plugin
