How to configure port forwarding in VirtualBox

  • Network -> Advanced -> Port forwarding -> host ip port(宿主机) guest ip port (虚拟机内系统)

硬盘管理

添加新分区到虚拟机

  1. Storge -> add Hard Dish -> create Disk 选择大小, 创建完成后, 启动虚拟机就能看到挂载了, 再改下 /etc/fstab 就能自动挂载了
