基础知识

ttf otf eot woff woff2

参考: Web 字体简介: TTF, OTF, WOFF, EOT & SVG

  1. TTF (TrueType Font) 字体格式是由苹果和微软为 PostScript 而开发的字体格式。
  2. OTF (OpenType Font) 由 TTF 演化而来，是 Adobe 和微软共同努力的结果。
  3. EOT (Embedded Open Type) 字体是微软设计用来在 Web 上使用的字体。
  4. WOFF (Web Open Font Format) 本质上是 metadata + 基于 SFNT 的字体（如 TTF、OTF 或其他开放字体格式）。
  5. WOFF2 是 WOFF 的下一代。 WOFF2 格式在原有的基础上提升了 30% 的压缩率。
  6. SVG (Scalable Vector Graphics font) 字体格式使用 SVG 的字体元素定义。
